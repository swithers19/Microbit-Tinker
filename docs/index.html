Embedded System? Sounds Scary/Boring.

Embedded systems are a complete computer system that perform certain things within a bigger system (mechanical or electronic). Essentially, its a bunch of code run on a special type of computer to do some of the following:

Spot (Boston Dynamics)

The most exciting embedded/Mechatronic system are robots. Here is a personal favourite of Boston Dynamic's Spot.

Ventilators

Medical devices such as ventilators, respirators ands anything with "smarts" processes vitals and determines what must be delivered to a patient. Due to that rather signifigant 2020 event, Medtronic have released the source code and drawings which can be found here.

Planes, Trains and Automobiles

Pretty much everything from your car to your washing machine has an embedded system. A challenge could be to find a home appliance that doesn't feature an embedded system.
